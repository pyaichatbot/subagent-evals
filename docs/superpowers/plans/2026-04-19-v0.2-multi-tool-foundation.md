## Section ID derivation — locked rule

All adapters that parse sections from a single file (codex `AGENTS.md`, `copilot-instructions.md`) must follow this rule:

- **Section heading**: the raw string after `## Agent:`, trimmed. Example: `## Agent: my reviewer` → `"my reviewer"`
- **`section` field on `DiscoveredAgent`**: set to the raw trimmed heading string.
- **`body` field on `DiscoveredAgent`**: set to the content between this heading and the next `## Agent:` heading (or end of file), trimmed.
- **Stable ID in auto mode** (set by `auto.ts`, not the adapter): `{format_adapter_id}:{relative-path-from-cwd}#{section}`. Example: `codex-md:packages/core/test/fixtures/codex-project/AGENTS.md#my reviewer`
- **Non-section files** (cursor-rules, windsurf, file-based codex): no `section` field. Auto-mode ID: `{format_adapter_id}:{relative-path-from-cwd}`.

Each adapter only sets `section` and `body`. The `id_override` is set exclusively by `auto.ts`.

  ```typescript
  import { existsSync } from "node:fs";
  import { readFile } from "node:fs/promises";
  import { resolve } from "node:path";

  import fg from "fast-glob";

  import type { DiscoveredAgent } from "../index.js";

  export async function discoverCodexAgents(input: {
    cwd: string;
    roots: string[];
  }): Promise<DiscoveredAgent[]> {
    const results: DiscoveredAgent[] = [];

    for (const root of input.roots) {
      const absoluteRoot = resolve(input.cwd, root);

      // File-based: .codex/agents/*.md — one agent per file
      const agentsDir = resolve(absoluteRoot, ".codex/agents");
      if (existsSync(agentsDir)) {
        const paths = await fg("**/*.md", { cwd: agentsDir, absolute: true });
        results.push(
          ...paths.map((path) => ({
            path,
            format_adapter_id: "codex-md" as const
            // section is intentionally absent for file-based agents
          }))
        );
      }

      // Section-based: AGENTS.md — multiple agents per file
      const agentsMd = resolve(absoluteRoot, "AGENTS.md");
      if (existsSync(agentsMd)) {
        const body = await readFile(agentsMd, "utf8");
        const sections = extractAgentSections(body);
        results.push(
          ...sections.map(({ heading, content }) => ({
            path: agentsMd,
            format_adapter_id: "codex-md" as const,
            body: content,
            section: heading  // raw trimmed heading string per the locked ID rule
          }))
        );
      }
    }

    return results;
  }

  function extractAgentSections(text: string): Array<{ heading: string; content: string }> {
    const lines = text.split("\n");
    const sections: Array<{ heading: string; content: string }> = [];
    let currentHeading: string | null = null;
    const currentLines: string[] = [];

    for (const line of lines) {
      const match = line.match(/^##\s+Agent:\s+(.+)$/i);
      if (match) {
        if (currentHeading !== null) {
          sections.push({ heading: currentHeading, content: currentLines.join("\n").trim() });
          currentLines.length = 0;
        }
        currentHeading = match[1].trim();
      } else if (currentHeading !== null) {
        currentLines.push(line);
      }
    }

    if (currentHeading !== null && currentLines.length > 0) {
      sections.push({ heading: currentHeading, content: currentLines.join("\n").trim() });
    }

    return sections;
  }
  ```

  import type { DiscoveredAgent } from "../index.js";

  export async function discoverCopilotAgents(input: {
    cwd: string;
    roots: string[];
  }): Promise<DiscoveredAgent[]> {
    const results: DiscoveredAgent[] = [];

    for (const root of input.roots) {
      const filePath = resolve(input.cwd, root, ".github/copilot-instructions.md");
      if (!existsSync(filePath)) continue;

      const body = await readFile(filePath, "utf8");
      const sections = extractAgentSections(body);

      if (sections.length > 0) {
        // Section-based: multiple agents defined in one file
        results.push(
          ...sections.map(({ heading, content }) => ({
            path: filePath,
            format_adapter_id: "copilot-instructions" as const,
            body: content,
            section: heading  // raw trimmed heading string per locked ID rule
          }))
        );
      }
      // Files with no ## Agent: sections are skipped — they are plain instructions,
      // not structured agent definitions, and would produce a useless zero-score agent.
    }

    return results;
  }

    const lines = text.split("\n");
    const sections: Array<{ heading: string; content: string }> = [];

    const currentLines: string[] = [];

    for (const line of lines) {
      const match = line.match(/^##\s+Agent:\s+(.+)$/i);
      if (match) {
        if (currentHeading !== null) {
          sections.push({ heading: currentHeading, content: currentLines.join("\n").trim() });
          currentLines.length = 0;
        }
        currentHeading = match[1].trim();
      } else if (currentHeading !== null) {
        currentLines.push(line);
      }
    }

    if (currentHeading !== null && currentLines.length > 0) {
      sections.push({ heading: currentHeading, content: currentLines.join("\n").trim() });
    }

    return sections;

  import { relative } from "node:path";

  import { discoverCodexAgents } from "./codex-md.js";
  import { discoverCopilotAgents } from "./copilot-instructions.js";
  import { discoverCursorAgents } from "./cursor-rules.js";
  import { discoverWindsurfAgents } from "./windsurf-config.js";
  // discoverClaudeAgents is internal to index.ts — call discoverAgents with format "claude-md"
  // to avoid a true circular reference at module init time.
  import { discoverAgents } from "../index.js";

  import type { DiscoveredAgent } from "../index.js";

  export async function discoverAutoAgents(input: {
    cwd: string;
    roots: string[];
    globs?: string[] | undefined;
  }): Promise<DiscoveredAgent[]> {
    // Run all adapters in parallel
    const [claudeAgents, codexAgents, copilotAgents, cursorAgents, windsurfAgents] =
      await Promise.all([
        discoverAgents({ ...input, format: "claude-md" }),
        discoverCodexAgents(input),
        discoverCopilotAgents(input),
        discoverCursorAgents(input),
        discoverWindsurfAgents(input)
      ]);

    const allAgents = [
      ...claudeAgents,
      ...codexAgents,
      ...copilotAgents,
      ...cursorAgents,
      ...windsurfAgents
    ];

    // Warn when 3+ adapter types are found (mixed repo may have dedup issues)
    const activeAdapters = new Set(allAgents.map((a) => a.format_adapter_id));
    if (activeAdapters.size >= 3) {
      process.stderr.write(
        `[subagent-evals] Warning: 3 or more agent formats detected ` +
          `(${[...activeAdapters].sort().join(", ")}). ` +
          `Set discovery.dedup: true and discovery.primary to suppress duplicates.\n`
      );
    }

    // Set stable qualified ID on every agent per the locked ID rule:
    // {format_adapter_id}:{relative-path-from-cwd}  (file-based)
    // {format_adapter_id}:{relative-path-from-cwd}#{section}  (section-based)
    return allAgents.map((agent) => ({
      ...agent,
      id_override: buildQualifiedId(agent, input.cwd)
    }));
  }

  function buildQualifiedId(agent: DiscoveredAgent, cwd: string): string {
    const rel = relative(cwd, agent.path).replace(/\\/g, "/");
    const base = `${agent.format_adapter_id}:${rel}`;
    return agent.section !== undefined ? `${base}#${agent.section}` : base;
  }

If vitest reports a circular dependency crash, the fix is to extract `discoverClaudeAgents` to `src/adapters/claude-md.ts` following the same pattern and import it directly instead.

  ````markdown
  # subagent-evals

  **The Codecov for markdown AI agents.**

  Lint, eval, score, and ship agents for Claude Code, Codex, GitHub Copilot, Cursor, and Windsurf.

  ```bash
  npx subagent-evals@latest lint .
  ```

  ## What it does

  `subagent-evals` evaluates markdown-defined AI agents at two layers:

  - **Static** — quality heuristics: trigger clarity, scope, output contract, tool policy, model spec
  - **Runtime** — assertion testing against fixture or live output

  Auto-detects all agent formats in your repo. Works with every major AI coding tool.

  ## Supported formats

  | Tool | Location | Format flag |
  |---|---|---|
  | Claude Code | `.claude/agents/*.md` | `claude-md` |
  | OpenAI Codex | `.codex/agents/*.md` or `AGENTS.md` | `codex-md` |
  | GitHub Copilot | `.github/copilot-instructions.md` | `copilot-instructions` |
  | Cursor | `.cursor/rules/*.mdc` | `cursor-rules` |
  | Windsurf | `.windsurf/rules/*.md` | `windsurf-config` |
  | Generic | Any YAML-frontmatter `.md` glob | `generic-frontmatter-md` |

  ## Quickstart

  ```bash
  # Zero-config: auto-detects all formats in your repo
  npx subagent-evals@latest lint .

  # Initialize config
  npx subagent-evals@latest init

  # Full eval with runtime assertions
  npx subagent-evals@latest eval .

  # Generate HTML report from saved results
  npx subagent-evals@latest report --input out/results.json --output out/report.html
  ```

  ## Configuration

  ```yaml
  # subagent-evals.config.yaml
  discovery:
    format: auto            # auto | claude-md | codex-md | copilot-instructions | cursor-rules | windsurf-config
    roots: [.]
    dedup: false            # set true to suppress duplicates in mixed repos
    primary: claude-md      # primary adapter when dedup: true
  outputs:
    json: out/results.json
    html: out/report.html
    junit: out/results.junit.xml
  thresholds:
    fail_below: 0.55        # exit code 1 if any agent scores below this
  ```

  | Score | Badge |
  |---|---|
  | 0.90+ | `certified` |
  | 0.75–0.89 | `strong` |
  | 0.55–0.74 | `usable` |
  | < 0.55 | `experimental` |

  - [`examples/windsurf-project`](examples/windsurf-project) — Windsurf rules
